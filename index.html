Ubuntu

Ubuntu — отличный вариант для тех, кто не хочет тратиться. Это свободная операционная система с открытым кодом, что делает ее достаточно привлекательной для самых разных пользователей. Она регулярно обновляется, отличается удобным интерфейсом и высоким уровнем безопасности.

РОСА

Роса  - это операционная система с интуитивно понятным интерфейсом, которая предназначена для широкого применения. Она отлично подойдет для каждого типового пользователя того же Windows, так как весьма проста, как в установке, так и в использовании.

РЕДОС

Антивирусная программа

 – программа, предназначенная для борьбы с компьютерными вирусами.
В своей работе эти программы используют различные принципы для поиска и лечения зараженных файлов.
Для нормальной работы на ПК каждый пользователь должен следить за обновлением антивирусов.
